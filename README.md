# Bank Marketing Campaign Analysis

## Overview
This project analyzes the Bank Marketing dataset to understand customer behavior and campaign performance.

## Tools
- Python (Pandas, Data Cleaning)
- SQL Server (Star Schema Modeling)
- Power BI (Dashboard Visualization)

## Workflow
1. Data Cleaning using Python
2. Data Modeling using SQL Server (Star Schema)
3. Data Visualization using Power BI

## Data Model
Star schema includes:
- dim_customer
- dim_contact_time
- dim_economic
- fact_marketing

## Dashboard Insights
- Customer distribution by age group
- Campaign success rate
- Loan status analysis
- Economic indicator trends

## Dataset
UCI Bank Marketing Dataset
